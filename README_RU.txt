Chess Helper v10 Stable Server Stockfish

Это исправление зависаний v9.

Почему v9 зависал:
- Render Free слабый.
- В v9 каждый ход запускал новый процесс Stockfish.
- Через несколько ходов сервер начинал тормозить.

Что изменено в v10:
- Stockfish запускается один раз и работает постоянно.
- Запросы идут по очереди.
- Есть авто-перезапуск движка при зависании.
- Для хода компьютера используется MultiPV=1.
- Для кнопки анализа используется MultiPV=3.
- Максимум времени ограничен 3 секундами для Render Free.

Что делать:
1. Замени файлы в GitHub на эту версию.
2. Render автоматически сделает новый deploy.
3. Открой тот же Render URL.
4. Выбирай 0.7 или 1 сек для максимальной стабильности.
